# URL

- `http://localhost:3000/?id=1`
  - 위와 같은 URL 주소는 아래와 같이 나눌수 있다.
  - http : pretocao
  - localhost : host(127.0.0.1)
  - 3000 : port
  - / : path(router)
  - id=1 : querystring
